# Configuration

Configuration files for this homelab are stored in this repository under [`config/`](config/) directory.

## Configuration files

| File              | Purpose                           |
| :-                | :-                                |
| bookmarks.yaml    | Static links                      |
| services.yaml     | Homelab services                  |
| widgets.yaml      | Status and information widgets    |
| settings.yaml     | Global Homepage behavior          |

## Configuration principles

- Services are grouped by function
- No credentials or secrets are stored in configuration files directly
